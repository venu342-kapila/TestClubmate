What happens when I delete a club?

All data is permanently deleted â€” transactions, polls, members, everything. Only the Owner can delete a club, and you must type the club name to confirm.

How does auto-update work?

The app has a service worker that checks for updates every 5 minutes. When an update is available, you'll see a "âœ¨ App updated!" toast and the app refreshes automatically. No reinstall needed.

ClubMates uses Google Sign-In and Firebase's security rules. Only club members can see their club's data. All communication is encrypted (HTTPS).
